## Image operations:
* Blob
* Contour
* Brightest pixel
* Timestamped stream
* Timestamped single frame

## Streaming functions:
* OSC (default port 7110)
* Mjpeg stream (default port 7111)
* Websockets (default port 7112)
* HTTP (default port 7113)

## Identify RPi by:
* random unique id
* hostname
